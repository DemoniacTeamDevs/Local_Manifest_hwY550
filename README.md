# Local_Manifest
for hwY550 mokee
=================================


Instructions
---------------

Initializing:

    git clone https://github.com/DemoniacTeamDevs/Local_Manifest_hwY550 -b master .repo/local_manifests

Sync up:

	repo sync --force-sync --force-broken
	
-------------
 
_Building_
---------------

First:

	cd ~/mokee

Second:

	. build/envsetup.sh

Third:

    brunch codename
    
Example:

    brunch y550
